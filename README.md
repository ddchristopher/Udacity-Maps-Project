# Neighborhood Map Project for Udacity Front End Developer Nanodegree
## 
by Dmitri Christopher

An app that displays my favorite places to visit in Venice, Italy.

Uses the Bootstrap 4, Knockout.js, JQuery, Google Maps API and Foursquare API.


## How to use:

### Option 1

1. Clone or download the repository.

2. Open the index.html file


### Option 2

Use the hosted version at https://ddchristopher.github.io/Udacity-Maps-Project/
